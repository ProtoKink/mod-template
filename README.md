# mod-template
A common template for our mods

## Before you start

It's a good idea to update `bc-deeplib` and `bc-stubs` to the latest version, just in case...

## Usage

First edit the `deeplib.config.js` and `package.json` files to your needs.

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm install
pnpm run build
